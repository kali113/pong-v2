# GitHub Pages Setup Instructions

Your GitHub Pages site is ready! Just need to enable it:

## Enable GitHub Pages

1. Go to: https://github.com/kali113/pong-v2/settings/pages

2. Under "Build and deployment":
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: /docs

3. Click **Save**

4. Wait 1-2 minutes for deployment

5. Your site will be live at: **https://kali113.github.io/pong-v2/**
